Quá trình học tập

Năm 1

Môn học quan trọng:

  • Nhập môn lập trình
  • Cấu trúc dữ liệu và giải thuật
  • Toán rời rạc

Dự án: Xây dựng ứng dụng quản lý sinh viên đơn giản bằng Java

Năm 2

Môn học quan trọng:

  • Lập trình hướng đối tượng
  • Cơ sở dữ liệu
  • Phát triển ứng dụng web

Dự án: Website bán hàng với HTML, CSS, JavaScript và PHP

Năm 3

Môn học quan trọng:

  • Công nghệ phần mềm
  • Phát triển ứng dụng di động
  • Trí tuệ nhân tạo cơ bản

Dự án: Ứng dụng đặt đồ ăn trên điện thoại với React Native

Năm 4

Môn học quan trọng:

  • Phát triển phần mềm chuyên nghiệp
  • Học máy và học sâu
  • Thiết kế UI/UX

Dự án: Hệ thống phân tích dữ liệu sử dụng MERN stack
